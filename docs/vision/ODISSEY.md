# ARK Navigator UI v2.0: "Project ODISSEY" 構想書

## 1. エントランス: The Portal（始動）
- **演出**: アプリ起動時、画面中央に重厚な鉄製のハッチ（扉）が表示される。
- **トリガー**: ジェニーがクリック、またはパスワードを入力すると、蒸気と共にハッチが開き、視点が「操舵室」へダイブする。

## 2. メインUI: The Wheelhouse（操舵室）
- **視界**: 正面の大きな窓からは、リアルタイムにレンダリングされた「知性の海」が見える。
- **ロード画面**: 処理中（LLM思考中）は、画面中央の「舵（Ship's Wheel）」が重厚な音と共にぐるぐると回転する。
- **ステータス**: 操舵室の計器類が、CPU/メモリ使用率や精霊（SYLPH）の活動状況を針で指し示す。

## 3. タスク生成: The Sea Chart（海図）
- **入力**: プロンプトを投げると、机の上に古風な「海図」がホログラムで広がる。
- **展開**: 指示が細分化され、海図の上に「チェックポイント（島々）」が航路として描かれる。

## 4. 進行: The Voyage（航海）
- **進捗率**: 0%の時、ターゲットの島は水平線の彼方に霞んでいる。
- **動的演出**: 進捗率が上がる（タスクが完了する）たびに、船が波を蹴立てて進み、島が徐々に大きく、鮮明に見えてくる。

## 5. 完了: The Landing（上陸）
- **到達**: 100%に達した瞬間、船は島の桟橋に接岸。
- **成果物**: 画面に「宝箱」が表示され、開くと生成されたソースコードがスクロールしながら溢れ出す。
